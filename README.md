# yt-dlp-webui
simple youtube wav downloader using yt-dlp

[try on HF Space](https://huggingface.co/spaces/Hev832/youtube_downloader)


#how to run locally?

step 1 :
````
git clone https://github.com/laynz28/yt-dlp-webui.git
````
step 2 :

```
pip install requirements.txt
```

step 3 : 

```
python app.py
```
